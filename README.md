# -greenfoot-teste-

Reprodução do greenfoot teste.

- Tela de seleção de jogadores;
- Suporta até 2 jogadores;

Jogador 1 - Darth Vader:

A = esquerda
D = direita
W = cima
S = baixo
R = soltar poder

Jogador 2 - Yoda:

left = esquerda
right = direita
up = cima
down = baixo
space = soltar poder
